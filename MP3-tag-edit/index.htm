MP3 Tag Editor - A fully free MP3 editor

Welcome to the about page of my application MP3 Tag Editor. Here I'll explain to you the quick run down of features and function's in the program. Here's a list of things I'll be talking to you about:

  • Opening files
    • Folder
    • One file
  • The drag and drop feature
  • Applying changes
    • Title, artist, album, genre, year
    • Image Covers
  • Playing songs
  • Changing UI colors

  1. Opening files

    2. You can open one file or a folder with the following steps.

    • One file

      • To open one file go to File/Open Song

    • Folder

      • To open a folder click on the Directory button

  2. The drag and drop feature

    3. You can drag and drop .mp3 or image files. When you drag and drop multiple images the last image by alphabetical order gets imported.

  3. Applying changes

      4. Remember to click on Apply when you are done.
    • Title, artist, album, genre, year

      • To choose the title, album and genre just type in the wished values into each text box. To choose the year click on the combo box and choose the wished value. The start year value start at 1950 and goes to the current year.

    • Image Covers

      • To choose a image cover just click on the Open Image button or drag and drop a image from you computer or from the web.

  4. Playing songs

    5. To play song's, press on the play button, to pause press the pause button. To switch between song press on the arrow buttons.

  5. Changing ui colors

    6. To change ui colors got to Tools/Settings and the following windows will appear.

    To change a color just click on the desired color rectangle. Then change the hue, saturation and brightness value by sliding the slider thumb or by manually inputing the values. The hue is measured from 0 to 360Â°. The saturation and brightness are measured from 0 to 100%. You can also just type in the 6 or 8 bit hex code. Just remember to press enter to apply the inputed value in the the text boxes. When you are done selecting the color, click on OK and the UI will be updated with the wished colors.
