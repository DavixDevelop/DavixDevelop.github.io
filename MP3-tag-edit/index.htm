MP3 Tag Editor - A fully free MP3 editor

By DavixDevelop

Welcome to the about page of my application MP3 Tag Editor. Here I'll explain to you the quick run down of fetures and function in the program. Here is a list of things I'll be talking to you about:

  • Opening files
    • Folder
    • One file
  • The drag and drop feature
  • Applying changes
    • Title, artist, album, genre, year
    • Image Covers
  • Playing songs

  1. Opening files

    2. You can open one file, multiple files or a folder with the following steps.

    • One file
      • To open one file go to File/Open Song

    • Folder

      • To open a folder click on the Directory button

  2. The drag and drop feature

    3. You can drag and drop .mp3 or image files. When you drag and drop multiple images the last image by alphabetical order gets inmported.

  3. Applying changes

    • Title, artist, album, genre, year

      • To choose the title, album, genre and year just type in the wished valuse into each text box.

    • Image Covers

      • To choose a image cover just click on the Open Image button or drag and drop a image.

  4. Playing songs

    5. To play song press on the play button, to pause press the pause button. To switch between song press on the arrow buttons.
