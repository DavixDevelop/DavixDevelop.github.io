It took a while, but I succeeded. You can now customize the colors of the ui to your own needs. Just go to Tool/Setting and you will see this.

To change a color just click on the desired color rectangle. Then change the hue, saturation and brightness value by sliding the slider thumb or by manually inputing the values. The hue is measured from 0 to 360Â°. The saturation and brightness are measured from 0 to 100%. You can also just type in the 6 or 8 bit hex code. Just remember to press enter to apply the inputed value in the the text boxes. When you are done selecting the color, click on OK.

You can also change the order by clicking on the combobox displayed in the top left picture and choosing the wished order. If you added a new file to the music folder just click on the refresh button dispay in the top middle picture. If you want the program to scan for files in subdirecotries just go to Tools/Setting and click on the toggle displayed in the top right picture. If you need to know how many .mp3 files do you have in a folder, there's a text block on the bottom right corner now. The program now also performers better as I got rid of the background worker and switched to a Async method instead. Settings are also now stored in the user congif file, rather then in .txt file in AppData, which your can now remove by typing in %appdata% in search and then going to DavixDevelop\MP3 Tag Edit and deleting UserInfo.txt if you have it.
